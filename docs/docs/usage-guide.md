# Usage Guide

This guide will walk you through the main features of the Vocaloid Rate application.

## Main Page (Dynamic Track Table)

The main page displays the latest Top 300 Vocaloid tracks. Here's how you can interact with it:

- **Sorting:** Click on column headers (e.g., Title, Producer, Rank) to sort the table by that column. Click again to reverse the sort order.
- **Filtering:** Use the filter options at the top of the table to narrow down the displayed tracks:
  - **Text Search:** Type in the search bar to filter by Title (English/Japanese), Producer, or Voicebank.
  - **Rating Status:** Filter to show only Rated, Unrated, or All tracks.
  - **Chart Status:** Filter by tracks currently On Chart or Expired/Archived.

## Personal Ratings & Notes

You can rate songs and add personal notes to keep track of your preferences:

- **Rating:** Click on the star icon next to a track to assign a rating from 1 to 10 stars.
- **Notes:** After rating, you can add optional notes to remember details about the song.

## Playlists

Create and manage custom playlists to organize your favorite tracks.

- **Creating a Playlist:** Navigate to the Playlists page and use the option to create a new playlist.
- **Adding/Removing Tracks:** From the main track table or rated tracks page, you can add a track to an existing playlist. You can also remove tracks from a playlist's edit page.
- **Reordering:** On the playlist edit page, use drag-and-drop to change the order of tracks within your playlist.
- **Import/Export:** Share your playlists or import others' using the import/export functionality.

## Rated Tracks Page

This page provides a dashboard view of all your rated songs and detailed statistics.

- **Statistics:** View your average and median ratings, and see your Top 10 favorite producers and voicebanks.
- **Rating Distribution Chart:** An interactive chart allows you to filter tracks by a specific score.

## Options Page

The Options page allows you to manage application settings and data.

- **Backup/Restore:** Backup your ratings and playlists to a JSON file, or restore them from a previously saved file.
- **Page Size:** Change default items shown per page count.
